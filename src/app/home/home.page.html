My Knowledge

JavaScript
JavaScript is my main programming language. I've been studying it since the beginning of 2022 until now (June 2024). I've practiced many concepts in my projects, as you will see in the next section. Most of these projects were developed primarily using pure JavaScript. Currently, I'm studying asynchronous JavaScript after spending several weeks using and testing APIs.
Node.Js
Node.js is the first JavaScript library that I started using. It is probably the easiest one, and I really loved it for being so smooth and clean.
TypeScript
TypeScript has impressed me since I discovered the language while searching for new languages to learn after delving deeper into JavaScript. I really like how it gives you confidence that your code will work, and not having dynamic types has helped me avoid small bugs when parsing types. I really want to learn more about it.
C++
I've had many programming logic classes in the past year, and most of them were using C++. I didn't like the structure because it was so repetitive and hard to remember, but after some time, it becomes easier. It's good for creating games, but it is not a language that I plan to use in the future.
Java
I have a difficult relationship with Java. I dislike it because it is so hard and confusing, and it gives errors for almost everything that is considered acceptable in other languages. However, I have to admit that it helps you write correct code when you follow its logic.
HTML
The best way to start in front-end development is not through a specific programming language, but it is very good to start this way. It probably won't make you give up and will show you the tip of the iceberg that is the programming universe.
CSS
I really admire those who know how to use CSS well. I do my best, but I know I wasn't born to do it. I'd say CSS is similar to HTML in that it is easy to begin with.
SCSS
A very useful CSS library, when using TypeScript, is a great alternative to traditional CSS.
GitHub
I have been posting my code on GitHub since 2021, and I love saving my code in the cloud. GitHub's functionalities, like GitHub Pages, are great and provide free hosting for your websites.
Figma
I started using Figma this year for school projects. I found it to be a really helpful and easy-to-understand tool for visualizing how your application will look.
Git
I'm studying more about Git and saving files locally, and it has already helped me when I accidentally deleted the code from one of my school projects.
Angular
I'll say that I'm in love with Angular. It's pretty, simple, clean, and very useful. I'm going to study more about this framework and delve deeper into its syntax.
